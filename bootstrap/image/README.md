# 圖片(Image)

![Bootstrap Image](./assets/image.png)

**類別**

* img-rounded (圓角)
* img-circle (圓框)
* img-thumbnail (加外框)
* img-responsive：圖片會佔其容器的全部寬度，而高度會按比例調整。

**圖文區塊**

```
<div class="thumbnail">
  <a href="/w3images/lights.jpg">
    <img src="/w3images/lights.jpg" alt="Lights" style="width:100%">
    <div class="caption">
      <p>Lorem ipsum...</p>
    </div>
  </a>
</div>
```
